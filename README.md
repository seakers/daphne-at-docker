# Daphne AT Docker

## Prerequisites

- Docker Desktop ([Download](https://www.docker.com/products/docker-desktop))
- Git

## Installation

1. Clone the repository:
```bash
git clone https://github.com/seakers/daphne-at-docker.git
cd daphne-at-docker
docker-compose up

```

## Access
Frontend: http://localhost
API: http://localhost:8000
